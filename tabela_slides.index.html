Como colocar esta página no GitHub Pages

1

Crie uma conta no GitHub (se ainda não tiver)

2

Crie um novo repositório com o nome "seuusuario.github.io" (substitua "seuusuario" pelo seu nome de usuário)

3

Faça upload deste arquivo HTML para o repositório (nomeie como "index.html")

4

Vá nas configurações do repositório, selecione a opção GitHub Pages e escolha o branch principal

5

Pronto! Sua página estará disponível em https://seuusuario.github.io

6

Compartilhe este link com seus clientes
